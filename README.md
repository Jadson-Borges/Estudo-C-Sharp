# 📘 Estudo C#

Repositório criado para armazenar exercícios, exemplos e anotações durante meus estudos da linguagem C# e da plataforma .NET.

## 🚀 Objetivo

Este projeto tem como finalidade praticar conceitos fundamentais e avançados de C#, incluindo:

- Sintaxe da linguagem
- Variáveis e tipos
- Estruturas condicionais
- Laços de repetição
- Métodos e funções
- Programação Orientada a Objetos (POO)
- Arrays e coleções
- Tratamento de exceções
- LINQ
- MVC
- APIs
- Entity Framework
- Boas práticas de desenvolvimento

## 🛠️ Tecnologias utilizadas

- C#
- .NET
- Visual Studio Code
- Git e GitHub

## 📚 Organização do repositório

Os conteúdos estão separados por temas e exercícios para facilitar o aprendizado e evolução nos estudos.

Exemplo:

```bash
/Exercicios
/POO
/MVC
/APIs
/LINQ
```

## 🎯 Objetivo pessoal

Este repositório faz parte da minha jornada de aprendizado em desenvolvimento backend com C# e .NET, buscando evolução constante através da prática.

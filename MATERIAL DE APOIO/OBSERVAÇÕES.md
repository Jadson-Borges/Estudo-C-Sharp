**Tipos referência vs tipos valor**

tipos referencias são conhecidos também como ponteiros pois a variável a nível de memoria é armazenada na stack que aponta para área da memoria HEAP que é onde realmente é armazenado o valor daquela variável



exemplo TV = /0x100358/ esse valor dentro dos parênteses significa que o endereço ondo o dado real está armazenado. 0x100358: /Smart TV Sansung/

o valor recebido é esse a variável somente aponta para o local na memoria onde o dado real está armazenado!
